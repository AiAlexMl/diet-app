## 0. איך מייצרים ריל חדש (פקודה אחת + עיבוד) — להמשך יעיל

הכל רץ מקומית עם playwright-core + chromium הקיים (בלי MCP, בלי restart). מ-`C:\Users\alexm\.claude\skills\web-to-reels`:

```bash
FF="scripts/node_modules/ffmpeg-static/ffmpeg.exe"
# 1) הקלטת הזרימה (סקריפט קנוני: חיטוב+כשר+ערב, בחירת מאכלים בין טאבים, החרגת עדשים)
node scripts/record-shapeat.mjs
# 2) בניית הבסיס (recipe מנצח: holds קריאים, גוזר blank פתיחה)
node scripts/make-reel.mjs --cap 1.2 --trim-start 1.0 --intro-hold 0.4 --out reels/reel-base-v2.mp4
# 3) payoff-first: מקדים ~2.2ש' של גלילת התפריט כ-hook (DUR = אורך הבסיס)
DUR=$("$FF" -i reels/reel-base-v2.mp4 2>&1 | grep -oE "00:00:[0-9.]+" | grep -oE "[0-9]+\.[0-9]+" | head -1)
T1=$(awk "BEGIN{print $DUR-5.0}"); T2=$(awk "BEGIN{print $DUR-1.2}")
"$FF" -y -i reels/reel-base-v2.mp4 -filter_complex \
"[0:v]trim=$T1:$T2,setpts=(PTS-STARTPTS)/1.7,format=yuv420p[t];[0:v]setpts=PTS-STARTPTS,format=yuv420p[f];[t][f]concat=n=2:v=1[o]" \
-map "[o]" -c:v libx264 -profile:v high -crf 18 -preset slow -movflags +faststart -an reels/reel-payoff-first-v2.mp4
```

- **לשנות בחירות** (מטרה/דיאטה/מאכלים): לערוך את `scripts/record-shapeat.mjs` (מסכים 3–6). מאכל מוחרג חייב להיות מקטגוריה שלא בחרנו בה מועדף (אחרת סתירה).
- **קצר/ארוך**: `--cap 0.9` מהדק (~20ש'), `--cap 1.4` מאריך. הלקח: על מסך סטטי אל תשתול still ארוך (נראה כתקלה) — dwell חי עדיף.
- **לקרוא תוצאה**: לחלץ פריימים — `"$FF" -ss <שניות> -i reels/reel-payoff-first-v2.mp4 -frames:v 1 f.png`.
