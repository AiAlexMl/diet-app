# Diet Application — בונה תפריט תזונה
גרסה 1.0

## מבנה הפרויקט

```
DietApp/
├── index.html     — מבנה HTML, 5 מסכים
├── style.css      — כל העיצוב
├── data.js        — מאגר מזון (כ-100 פריטים) + תבניות שעות
├── app.js         — לוגיקה: חישובי מאקרו + אלגוריתם תפריט
└── ui.js          — ממשק: ניווט, רינדור, אירועי משתמש
```

## הרצה
פתח את `index.html` בדפדפן — לא נדרש שרת.

## זרימת המשתמש
1. פרטים אישיים (גיל/גובה/משקל/מין/מטרה)
2. העדפות תזונה + זמן אימון
3. סימון מאכלים מועדפים ❤️
4. סימון מאכלים מוחרגים 🚫
5. תפריט יומי מלא עם מאקרו

## לוגיקת חישוב
- BMR: Harris-Benedict 1919
- RMR: BMR × 1.2
- חיטוב: גירעון דינמי — min(500, RMR×20%)
- מסה: RMR + 300 קל׳
- רצפת קלוריות: 1500 (גברים) / 1200 (נשים)
- חלבון: משקל × 2 (BMI ≥30 → לפי משקל BMI 25)
- שומן: 20% מקלוריות (רצפה 25g גברים / 40g נשים)
- פחמימות: שאר הקלוריות
- יישור מאקרו לתפריט (נמדד): חלבון ~±3%, שומן ~±4%, קלוריות ~±4% (מסה ~−9%)

## כללים תזונתיים
- בוקר: חלבי/ביצים (לבשרים אין; לטבעונים לחם+ממרח)
- פחמימה חמה: רק עם בשר/דג חם (לא טונה שימורים)
- סלט: מינימום 2 ירקות + שמן זית
- ביצים: עד 2 ביום
- טונה: עד קופסה אחת ביום
- קוטג': עד קופסה אחת במנה
- ללא גלוטן: מחריג חיטה/שיפון/שיבולת שועל; לחם+פסטה ללא גלוטן ייעודיים
- קטנייה אינה "מנת חלבון" לאוכלי-כול (משמשת כך רק לטבעונים)
